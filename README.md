# БоберЧат

Мини-соцсеть + чат для узкого круга (лента с модерацией постов, профили, личные чаты).

Стек: PHP + MySQL, хостинг InfinityFree, автодеплой через GitHub Actions (FTP).

Статус: в разработке.

## Структура проекта

```
/public        — корень сайта (заливается на хостинг)
/public/api    — PHP-эндпоинты
/public/assets — css/js/img
/sql           — схема базы данных
```

## Ветки

- `main` — стабильная версия
- `beta-dev` — разработка новых фич
